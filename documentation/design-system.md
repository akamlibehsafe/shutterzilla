# ShutterZilla Design System

This document outlines the design system for the ShutterZilla platform, providing a consistent visual language and component library for implementation.

## 1. Color Palette

The color palette is designed to be clean, modern, and accessible, with a primary teal accent.

| Color | Hex | Usage |
|---|---|---|
| **Primary Teal** | `#0d9488` | Buttons, links, active states, highlights |
| **Slate Gray** | `#334155` | Body text, headings, dark backgrounds |
| **Light Gray** | `#f8fafc` | Page backgrounds |
| **White** | `#ffffff` | Card backgrounds, text on dark backgrounds |
| **Gray 300** | `#e2e8f0` | Borders, dividers |
| **Success** | `#16a34a` | Success messages, online indicators |
| **Warning** | `#f59e0b` | Warning messages, pending states |
| **Error** | `#dc2626` | Error messages, danger zones |
| **Mercari Red** | `#ef4444` | Mercari source badge |
| **Buyee Amber** | `#f59e0b` | Buyee source badge |
| **eBay Blue** | `#3b82f6` | eBay USA source badge |
| **JD Purple** | `#8b5cf6` | JD Direct Auctions source badge |

## 2. Typography

We use a clean, sans-serif font for readability.

- **Font Family:** Inter (or system sans-serif)
- **Headings:** `font-weight: 700`
- **Body:** `font-weight: 400`
- **Base Font Size:** 16px

| Element | Font Size | Font Weight |
|---|---|---|
| `h1` | 32px | 700 |
| `h2` | 24px | 700 |
| `h3` | 20px | 700 |
| `h4` | 18px | 700 |
| Body | 16px | 400 |
| Small | 14px | 400 |

## 3. Spacing

A consistent spacing scale is used for padding, margins, and gaps.

| Name | Value |
|---|---|
| `xs` | 4px |
| `sm` | 8px |
| `md` | 16px |
| `lg` | 24px |
| `xl` | 32px |
| `2xl` | 48px |
