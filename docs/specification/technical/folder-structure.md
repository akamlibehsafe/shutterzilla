# ShutterZilla Folder Structure for AI Project Docs

This document describes the folder structure adopted for the ShutterZilla project by using the AI-Docs-Toolkit of it developers

## Folder Structure

docs/
├── README.md                    # Main index (updated)
├── guides/                      # Process & workflow docs
│   ├── automation.md
│   ├── quick-reference.md
│   ├── documentation-workflow.md
│   └── migration/              # Migration docs (archived)
├── project/                     # Project-specific docs
│   ├── ai-context.md
│   ├── runbook.md
│   ├── decisions/              # ADRs
│   └── releases/               # Release notes
├── specification/              # Technical specs
│   ├── design/                 # Design specs
│   ├── technical/             # Tech stack, data models
│   ├── implementation/         # Implementation docs
│   └── functional/            # Functional requirements
├── mockups/                    # Design mockups
│   ├── current/              # Latest version (currently v2)
│   ├── v1/                   # Version 1 (archived)
│   └── v2/                   # Version 2 (will be archived when v3 is created)
└── assets/                    # Bra
