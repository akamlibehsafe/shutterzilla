# ShutterZilla Logo Files

This package contains two versions of the ShutterZilla logo in multiple formats and resolutions.

## Logo Versions

### Version 1: All Gray Aperture
- **Description**: Clean, professional look with all aperture blades in gray (#334155)
- **Best for**: Formal documents, business cards, letterheads, professional presentations
- **Folder**: `v1_all_gray/`

### Version 2: Accent Blade
- **Description**: Subtle teal accent on the top-right aperture blade for visual interest
- **Best for**: Website headers, app interfaces, marketing materials, social media
- **Folder**: `v2_accent_blade/`

## Brand Colors

- **Slate Gray**: `#334155` (used for "SHUTTER" text, circle outline, and most aperture blades)
- **Teal**: `#0d9488` (used for "ZILLA" text and accent elements)

## File Formats & Usage

### SVG (Vector Format)
- **File**: `shutterzilla_v1_all_gray.svg` or `shutterzilla_v2_accent_blade.svg`
- **Usage**: Best for print materials, scalable graphics, and when editing is needed
- **Benefits**: Infinite scalability without quality loss, small file size, editable in vector software
- **Recommended for**: Business cards, letterheads, large format printing, signage, vehicle wraps

### PNG (Raster Format)

#### 4K Resolution (3840px width)
- **File**: `shutterzilla_v1_4k.png` or `shutterzilla_v2_4k.png`
- **Usage**: High-resolution print materials, large displays, billboards
- **Recommended for**: Trade show banners, posters, large format prints

#### 2K Resolution (2048px width)
- **File**: `shutterzilla_v1_2k.png` or `shutterzilla_v2_2k.png`
- **Usage**: High-quality web graphics, presentations, brochures
- **Recommended for**: Website hero images, PowerPoint presentations, PDF documents

#### 1K Resolution (1024px width)
- **File**: `shutterzilla_v1_1k.png` or `shutterzilla_v2_1k.png`
- **Usage**: Standard web graphics, email signatures, social media headers
- **Recommended for**: Website headers, LinkedIn banners, email newsletters

#### 512px Resolution
- **File**: `shutterzilla_v1_512.png` or `shutterzilla_v2_512.png`
- **Usage**: Web thumbnails, app icons (large), social media profile images
- **Recommended for**: Facebook profile pictures, website favicons (large), app store listings

#### 256px Resolution
- **File**: `shutterzilla_v1_256.png` or `shutterzilla_v2_256.png`
- **Usage**: Small web icons, mobile app icons, favicons
- **Recommended for**: Mobile app icons, website favicons, small UI elements

## Usage Recommendations

### Web & Digital
- **Website Header**: Use 1K or 2K PNG, or SVG for best quality
- **Favicon**: Use 256px PNG or create ICO from it
- **Social Media Profile**: Use 512px PNG
- **Email Signature**: Use 512px or 1K PNG
- **Web App Interface**: Use SVG or 1K PNG

### Print & Stationery
- **Business Cards**: Use SVG or 4K PNG
- **Letterhead**: Use SVG or 4K PNG
- **Brochures**: Use SVG or 2K-4K PNG depending on size
- **Posters/Banners**: Use SVG or 4K PNG
- **Merchandise**: Use SVG for best quality

### Presentations & Documents
- **PowerPoint/Keynote**: Use 1K-2K PNG or SVG
- **PDF Documents**: Use SVG or 2K PNG
- **Word Documents**: Use 1K PNG

## Technical Specifications

- **Aspect Ratio**: 4:1 (landscape orientation)
- **Color Space**: RGB (for digital), convert to CMYK for print
- **Background**: Transparent (PNG) or white (as shown)
- **Font**: Bold sans-serif (Arial/Helvetica style)

## Notes

- Always maintain the aspect ratio when resizing
- For print materials, ensure minimum 300 DPI resolution
- Use SVG whenever possible for maximum quality and flexibility
- Keep a backup of all original files
- For web use, optimize PNG files for faster loading if needed
